---------
- Responsive design (mobile, tablet, desktop)
- Accessible: ARIA roles, skip link, alt text, keyboard navigation
- FAQ accordion with keyboard support
- Contact form with client-side validation + confirmation message
- Sticky navigation header with mobile hamburger menu
- WCAG 2.1 AA compliant colour contrast
- Semantic HTML5 structure
- CSS custom properties (variables) for consistent theming
- Lazy loading attributes on images
- Meta description tags on all pages
- Skip navigation link for screen readers
- All pages: consistent header, navigation and footer
